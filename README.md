## Installation

Execute the following commands:<br>
`cd ~/ros_ws/src`<br>
`git clone https://github.com/angiecodinghub/panda_openai.git`<br>
`cd ~/ros_ws`<br>
`catkin_make`<br>
`source devel/setup.bash`<br>
`rosdep install panda_openai`<br>
